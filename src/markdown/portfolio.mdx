### WASA Rockoon Project
(Aug 2023 - Present)
| ![WASA Rockoon Project](/home/WASARockoonProject.jpg)  |
| --- |
| Aerospace Engineering, Embedded System, Wireless Engineering |
| Rockoonはrocketとballoonを組み合わせた造語で，気球による空中発射式のロケットを指す。ロケットを吊り下げた気球を打ち上げ，成層圏まで到達したあとロケットを分離して発射する。早稲田大学の宇宙系ものづくりサークルWASAに所属し，高度100kmの宇宙空間到達を目標にして技術開発を進めている。主に電装まわりを担当。T系（通信系）の回路基板やアンテナを開発中。|

### AUDAS
(July 2020 - Nov 2020)
| ![AUDAS](/home/AUDAS.jpg)  |
| --- |
| Research, Embedded System |
| AUDAS(Automated Underwater Data Acquisition System:自動水中データ収集システム)は，大気と水中環境の温度や湿度といったテレメトリを，全自動で同時にモニタリングするシステム。水中環境の未来予測を目標としていて，将来的に水中の調査や漁業に役立てる。自分はPMとして，調査，研究開発，論文執筆を総括した。自分達の持っている技術を使って研究を行う難しさや論文を執筆する難しさを学んだ。COVID-19に伴いこのプロジェクトは終了した。|
